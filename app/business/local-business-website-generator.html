What It Does

Generates complete, professional websites for local businesses using AI. Input basic business information and receive a fully designed, mobile-responsive website.

Problem It Solves

Many small local businesses need web presence but can't afford custom development. This democratizes professional web design through AI automation.

Key Features

  • AI-generated content from business description
  • Industry-specific templates
  • Automatic SEO optimization
  • Mobile-responsive design
  • One-click deployment

Tech Stack

Python OpenAI Next.js Vercel

API Costs

  • OpenAI API - Paid (~$0.002-0.03/1K tokens)
  • Vercel Hosting - Free tier available

Why I Built This

I saw many small businesses in my area with no online presence. This tool makes it easy for anyone to get a professional website without technical knowledge.
