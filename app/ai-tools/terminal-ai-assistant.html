What It Does

A terminal-based AI assistant that helps developers with coding tasks, debugging, and system administration directly from the command line.

Problem It Solves

Context switching between terminal and browser to ask AI questions breaks flow. This brings AI assistance directly into the developer's natural environment.

Key Features

  • Natural language commands in terminal
  • Code generation and explanation
  • Shell command suggestions
  • Context-aware responses based on current directory
  • History and conversation memory

Tech Stack

Python Ollama LLaMA

API Costs

  • Ollama + LLaMA - Free (runs locally on your machine)

No API costs - everything runs locally.

Why I Built This

I wanted an AI assistant that fits into my terminal-centric workflow without needing to open a browser or switch contexts.
