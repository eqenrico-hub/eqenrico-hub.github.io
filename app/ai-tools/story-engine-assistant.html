What It Does

An interactive storytelling system that uses AI to generate dynamic narratives based on user choices. Creates branching storylines with consistent characters and plot development.

Problem It Solves

Traditional choose-your-own-adventure stories have limited paths. This creates truly dynamic stories that adapt to any choice while maintaining narrative coherence.

Key Features

  • Dynamic story generation
  • Character memory and consistency
  • Multiple genre support (fantasy, sci-fi, mystery)
  • Save and resume stories
  • Story export to various formats

Tech Stack

Python LangChain GPT API

API Costs

  • OpenAI GPT API - Paid (~$0.002-0.03/1K tokens depending on model)

Why I Built This

I love interactive fiction and wanted to see how AI could create truly unlimited storytelling possibilities while keeping narratives engaging and coherent.
