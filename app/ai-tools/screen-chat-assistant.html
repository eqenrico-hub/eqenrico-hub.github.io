What It Does

An AI assistant that can capture and analyze your screen content, allowing you to ask questions about what you're looking at or get help with visible tasks.

Problem It Solves

Describing visual content to AI is tedious and error-prone. This lets the AI see exactly what you're seeing for more accurate assistance.

Key Features

  • Screen capture and analysis
  • OCR for text extraction
  • Visual element identification
  • Context-aware responses
  • Support for code, documents, and applications

Tech Stack

Python GPT-4 Vision PyAutoGUI

Why I Built This

Sometimes the fastest way to explain a problem is to show it. This bridges the gap between visual context and AI assistance.
