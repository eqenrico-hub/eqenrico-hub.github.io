What It Does

A platform to aggregate cultural events, exhibitions, and performances from Italian venues into a unified, searchable interface.

Problem It Solves

Cultural events in Italy are advertised across fragmented platforms - museum websites, theater pages, local event boards. Finding what's happening requires checking dozens of sites.

Planned Data Sources

  • Museum and gallery websites
  • Theater and concert venues
  • Local event calendars
  • Cultural institution APIs

Planned Features

  • Multi-venue event scraping
  • Category filtering (art, music, theater)
  • Location-based search
  • Calendar integration
  • Personalized recommendations

Tech Stack

Next.js TypeScript TBD (scraping)

Status

Currently just a Next.js frontend boilerplate. Scraping logic and data sources not yet implemented.

Why I'm Building This

I love attending cultural events but find it frustrating to discover what's happening across different venues and cities.
