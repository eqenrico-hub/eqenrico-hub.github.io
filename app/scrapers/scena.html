What It Does

Scrapes cultural events, exhibitions, and performances from Italian venues and aggregates them into a searchable database with personalized recommendations.

Problem It Solves

Cultural events in Italy are advertised across fragmented platforms. This creates a unified view of what's happening in art, music, theater, and exhibitions.

Key Features

  • Multi-venue event scraping
  • Category filtering (art, music, theater, etc.)
  • Location-based search
  • Calendar integration
  • Personalized event recommendations

Tech Stack

Python Scrapy PostgreSQL React

Why I Built This

I love attending cultural events but found it frustrating to discover what's happening. This puts all the information in one place.
