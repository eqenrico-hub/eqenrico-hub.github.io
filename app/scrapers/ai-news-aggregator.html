What It Does

A customizable news research tool that uses OpenAI's web search capabilities to find and summarize the latest news on any topic you specify.

Problem It Solves

Staying informed on fast-moving topics requires constant searching. This automates the research and delivers AI-summarized updates with source links.

Key Features

  • Custom topic prompts (enter any subject)
  • OpenAI web search integration
  • AI-powered summarization with bullet points
  • Source link extraction (3-8 links per query)
  • Simple web interface

Tech Stack

HTML/CSS/JS OpenAI API

Why I Built This

I wanted a quick way to get AI-curated news summaries on specific topics without manually searching multiple sites.
