What It Does

A client-side web app for AI-powered news research. Enter any topic, get summarized results with sources, then export to Obsidian or generate a podcast with text-to-speech.

Problem It Solves

Staying informed on fast-moving topics requires constant searching. This automates research, delivers AI-summarized updates with source links, and lets you build a searchable knowledge base in Obsidian or listen to your research as podcasts.

Key Features

  • Multi-Widget Dashboard - Track multiple topics simultaneously
  • OpenAI Web Search - Real-time web search for fresh information
  • AI Summarization - Bullet points with 3-6 source links
  • Obsidian Export - Download as markdown with frontmatter and wikilinks
  • Podcast Generation - Convert results to audio with 6 voice options
  • Domain Filtering - Restrict searches to specific websites
  • Local Storage - API key and widgets persist in browser

Export Options

Format What You Get
Obsidian Markdown YAML frontmatter, tags, wikilinks to other notes, source list
Audio Podcast MP3 file with natural narration, auto-downloads after generation

  • Alloy - Neutral, clear

Tech Stack

API Costs

  • OpenAI GPT-4o-mini - ~$0.00015/1K input, $0.0006/1K output (research)
  • OpenAI Web Search - Included with Responses API
  • OpenAI TTS HD - ~$0.03/1K characters (podcast)

Typical research query: ~$0.01. Podcast generation: ~$0.05-0.10 per audio.

How to Use

  1. Open the HTML file in your browser
  2. Enter your OpenAI API key and save
  3. Create widgets for topics you want to track
  4. Click "Run" to research or "Refresh All" for all widgets
  5. Export results to Obsidian or generate podcasts

Why I Built This

I wanted a quick way to get AI-curated news on specific topics without manually searching. The Obsidian export builds a permanent knowledge base over time, and the podcast feature lets me catch up on research during commutes.
