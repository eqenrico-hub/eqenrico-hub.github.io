What It Does

A fully autonomous system that discovers YouTube channels in your fields of interest, analyzes videos for product opportunities, and surfaces market gaps - running continuously without intervention.

Problem It Solves

Product ideas are hidden in YouTube comments, video content, and creator discussions. Manual research is time-consuming. This automates the discovery of unmet needs and product opportunities.

Data Sources

  • YouTube - Channel discovery, video transcripts, comments

Key Features

  • Autonomous channel discovery by field (audio production, trading, etc.)
  • Video transcript analysis for pain points
  • Three modes: Full (API), Local (Ollama), Basic (no API)
  • Web dashboard for monitoring
  • Docker-based deployment

Tech Stack

Python YouTube API OpenAI/Anthropic/Ollama PostgreSQL Redis Docker

API Costs

  • YouTube Data API - Free (10,000 units/day quota)
  • OpenAI API - Paid (~$0.03/1K tokens) - optional
  • Anthropic Claude - Paid (~$0.01-0.08/1K tokens) - optional
  • Ollama/LLaMA - Free (runs locally) - optional

Can run in Basic mode with zero API costs.

Why I Built This

I wanted to discover product opportunities in niches I care about without spending hours watching YouTube videos and reading comments.
