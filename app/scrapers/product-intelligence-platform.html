What It Does

A premium SaaS platform for real-time market analysis, competitive intelligence, and AI-powered idea generation. Aggregates data from multiple tech/startup sources.

Problem It Solves

Understanding the competitive landscape requires monitoring multiple platforms constantly. This consolidates product launches, discussions, and trends into one dashboard.

Data Sources

  • Product Hunt - New product launches
  • GitHub - Open source projects and trends
  • Hacker News - Tech discussions and launches
  • Reddit - Community discussions and feedback

Key Features

  • Real-time multi-source scraping
  • AI-powered product analysis (Claude)
  • Visual analytics and dashboards
  • Smart idea generation from market gaps
  • Legal risk assessment (patents, copyright)
  • Automated alerts for opportunities

Tech Stack

FastAPI React PostgreSQL Redis Playwright Claude AI

Why I Built This

I wanted a single dashboard to track what's being built in tech, spot trends early, and find gaps in the market.
