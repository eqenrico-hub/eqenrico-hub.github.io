What It Does

An AI email assistant that can access your local files to help draft emails with relevant context, find attachments, and manage email workflows.

Problem It Solves

Email drafting often requires referencing local documents or finding the right attachments. This connects email assistance with file system access.

Key Features

  • AI-powered email drafting
  • Local file search for attachments
  • Context extraction from documents
  • Email template management
  • Smart reply suggestions

Tech Stack

Python Claude API Gmail API FUSE

Why I Built This

Email is a constant part of work, and having AI assistance that understands my local context makes responses more accurate and relevant.
