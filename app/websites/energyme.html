What It Does

A full marketing website for EnergyMe, an Italian startup building open-source energy monitoring hardware. The site showcases their product line (EM-Home, EM-Intelligence, EM-Industry), explains the technology through an animated flow diagram, and includes a shop, team page, and company story.

Key Features

  • Animated "How It Works" flow diagram with particle stream and interactive tooltips
  • Interactive world map showing global device network
  • Scroll-driven timeline on "Our Story" page
  • Flash-free scroll animations (CSS transitions + IntersectionObserver, not Framer Motion)
  • Product cards with hover effects and glass morphism design
  • Full Terms and Conditions page
  • Mobile responsive across all sections

Tech Stack

Next.js 16 React TypeScript Tailwind CSS v4 Framer Motion Netlify

Design Approach

Dark theme with green (#22c55e) accents matching EnergyMe's brand. Glass morphism cards, SVG animated icons, and a particle background in the hero. All scroll-reveal animations use pure CSS transitions with IntersectionObserver to avoid SSR hydration flash issues common with Framer Motion in Next.js.

Built For

EnergyMe S.r.l. — an Italian startup founded by three engineers building open-source, non-invasive energy monitors. OSHWA Certified (IT000025).
