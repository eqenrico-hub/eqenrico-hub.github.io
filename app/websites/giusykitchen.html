What It Does

GiusyKitchen is a recipe website featuring AI-powered chef assistants, an interactive cooking mode with step-by-step guidance, and photo analysis to identify ingredients from pictures.

Problem It Solves

Traditional recipe websites are static and don't adapt to user needs. GiusyKitchen provides an interactive cooking companion that helps users through each step, answers questions in real-time, and can suggest recipes based on ingredients you have.

Key Features

  • AI Chef Assistants - Multiple chef personalities to guide you
  • Cooking Mode - Step-by-step hands-free cooking guidance
  • Photo Analysis - Take a photo of ingredients, get recipe suggestions
  • Shopping List - Auto-generated from recipes
  • User Profiles - Save favorites and track cooking history

Tech Stack

React TypeScript Firebase OpenAI API Node.js

Why I Built This

My family has amazing recipes passed down through generations. I wanted to preserve them digitally and make cooking more accessible with AI assistance - especially for people who are intimidated by cooking or need extra guidance.
