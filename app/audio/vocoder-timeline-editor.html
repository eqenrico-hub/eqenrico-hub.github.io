What It Does

A visual timeline editor for arranging audio clips and visual events with precise timing. Drag clips to position them, adjust fades with handles, preview playback in real-time, and export timing code for implementation. Built specifically for designing the Band Ritual vocoder demo experience.

Problem It Solves

When building complex audio/visual experiences with narration, music, and animated elements, it's nearly impossible to describe exact timings in words. This tool lets you visually arrange everything on a timeline, hear how it sounds, see when visual elements appear, and then export the precise timing data for implementation.

Key Features

  • Multi-track Timeline - Separate tracks for vocals, synth, vocoder, visual events, and transitions
  • Waveform Display - See audio waveforms extracted from actual files
  • Draggable Clips - Move and resize audio events by dragging
  • Fade Handles - Drag cyan dots to adjust fade in/out times visually
  • Real-time Preview - Play timeline with actual audio and see playhead position
  • Visual Preview Panel - See VOICE/SYNTH boxes animate as they would in the final experience
  • Morph Indicator - Shows DRY to WET morph progress percentage
  • Undo/Redo - Cmd+Z / Cmd+Shift+Z with 20-step history
  • Marquee Selection - Select multiple clips with rectangle drag
  • Drag & Drop Audio - Add new audio files by dropping onto tracks
  • Export Timing Code - Copy structured timing data for Claude to implement

How It Works

AUDIO LOADING ├── Fetches WAV files via HTTP ├── Decodes to AudioBuffer └── Extracts waveform data for visualization ↓ VISUAL TIMELINE ├── Canvas-drawn waveforms ├── CSS-positioned event blocks ├── Draggable fade handles └── Red playhead with seek ↓ PLAYBACK ENGINE ├── Web Audio API scheduling ├── BufferSource per audio event ├── GainNode for volume/fades └── requestAnimationFrame for UI sync ↓ EXPORT ├── Structured timing object ├── Absolute start/end times └── Fade durations for each clip

Why I Built This

I was building a vocoder demo experience for the Band Ritual website with 4 vocal narration clips, synth audio, animated boxes, and a dry-to-wet morph transition. Trying to describe exact timings ("the synth should fade in 0.96 seconds starting at 8.80s") was tedious and error-prone. This tool lets me visually arrange everything, hear it play back, and export the exact timings for implementation.
