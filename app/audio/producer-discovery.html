What It Does

Discovers underground electronic music producers (2K-5K SoundCloud followers) by mining the social graph of artists you already know. Uses seed artists to find who they follow and who follows them, filtering to producers with tracks in your target follower range.

Problem It Solves

Finding emerging color bass, future riddim, and experimental bass producers is tedious. Forums are outdated, playlists favor big names, and manual searching takes forever. This tool automates discovery by mining who your favorite producers follow, revealing underground artists you'd never find manually.

Key Features

  • Mines "Following" lists of seed artists
  • Finds followers who also produce (have tracks)
  • Filters by follower count (default 2K-5K)
  • STRICT Genre Filtering - Only includes artists with explicit riddim/dubstep/color bass tags
  • Entity Detection - Automatically excludes labels, magazines, radio shows, bands
  • Weighted Track Analysis - Analyzes up to 20 tracks per artist for genre accuracy
  • Confidence scoring (verified/high/medium/low based on network connections)
  • Recursive/deep discovery mode (uses found producers as new seeds)
  • SQLite cache for persistent results across runs
  • Gmail integration for automated outreach
  • CSV export for sorting and filtering

Tech Stack

Python 3.11+ httpx Rich CLI SQLite SoundCloud Internal API Gmail API

System Architecture

SEED ARTISTS (known producers) ↓ DISCOVERY LAYER ├── Mine "Following" lists (200 max) └── Fetch user profiles + tracks ↓ MULTI-STAGE FILTERING ├── Entity filter: Reject labels, magazines, radio, bands ├── Follower filter: 2K-5K range only ├── Track filter: Must have 2+ original tracks ├── Genre filter: Analyze 20 tracks for explicit tags │ ├── TARGET: riddim, dubstep, color bass, future bass │ └── EXCLUDE: techno, house, DnB, trance, etc. └── Weighted scoring: genre field > tags ↓ CONFIDENCE SCORING ├── verified: Explicit genre tag found ├── high: 3+ verified connections ├── medium: 2 verified connections └── low: 0-1 (excluded from emails) ↓ CACHE (SQLite) + CSV EXPORT + GMAIL OUTREACH
