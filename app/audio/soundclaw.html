What It Does

SoundClaw is a VST3 plugin that embeds an AI chat interface directly inside your DAW. You type natural language prompts like "make the bass darker" or "add reverb to track 3" and SoundClaw translates them into actual DAW actions — controlling Ableton Live parameters, Serum/Serum 2 synthesizer settings, and plugin chains in real time.

Problem It Solves

Music producers spend hours tweaking knobs and navigating menus. SoundClaw turns sound design into a conversation: describe what you want and the AI figures out which parameters to change. It bridges the gap between creative intent and technical execution, especially for producers who know what they want to hear but not which knob to turn.

Key Features

  • 124 DAW commands — transport, mixing, routing, plugin control, loop expansion
  • Claude API agent loop — multi-step reasoning with tool use for complex requests
  • Ableton Live integration — via custom Remote Script (Python, installed in User Library)
  • Serum / Serum 2 control — direct parameter manipulation via OSC or MIDI CC
  • Chat UI inside the plugin — no browser, no external app, embedded in the DAW workflow
  • Loop expansion — AI-assisted arrangement from loops to full sections

Tech Stack

JUCE (C++) VST3 / AU Claude API Python (Remote Script) Ableton Live SDK OSC Protocol

Why I Built This

I produce music daily and I got tired of the disconnect between knowing what a track needs and spending 20 minutes finding the right parameter. SoundClaw started as an experiment ("what if I could just tell my DAW what to do?") and evolved into a full agent system with 124 commands. The goal is to make the DAW feel like a collaborator, not a tool.
