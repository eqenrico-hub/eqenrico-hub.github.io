What It Does

LevelLens is a VST3/AU plugin that automatically normalizes the loudness of individual audio events (hits, stabs, notes) using constant gain per event - without changing the waveform shape or using compression.

Problem It Solves

When layering samples or working with inconsistent recordings, volume levels vary wildly between hits. Traditional solutions like compression change the sound character. LevelLens maintains the original waveform while evening out levels.

Key Features

  • Event Detection - Automatically detects sound events based on energy
  • Per-Event Gain - Applies constant gain multiplier per event
  • Waveform Preservation - No compression or dynamic shaping
  • Smooth Transitions - Cosine ramps prevent clicks
  • Lookahead - Protects transients by pre-calculating gain

Tech Stack

C++ JUCE Framework VST3 SDK DSP

System Architecture

Audio Input → Event Detector → Gain Calculator → Gain Applier → Output ↓ ↓ Energy Analysis Lookahead Buffer Threshold Check Smooth Transitions

Why I Built This

While producing music, I needed a way to normalize drum hits without affecting their punch. Compressors squash the sound, and manual gain automation is tedious. LevelLens automates what I was doing by hand.
