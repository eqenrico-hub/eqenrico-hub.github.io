What It Does

LAYERS is a VST3/AU multi-sample instrument plugin by Ritual Audio Labs. Users stack audio samples as colored visual layers, draw a target output curve, and the engine automatically mixes them to match that curve using a dual-priority system: layer priority (volume/headroom order) and frequency priority (spectral ownership).

Problem It Solves

Multi-sample mixing has always been chaos — layers competing for headroom, frequencies clashing, and no structure to the combined output. Traditional sample layering requires manual volume automation, EQ carving, and constant A/B testing. LAYERS solves this by letting producers draw the desired output shape and having the engine intelligently distribute each sample's contribution.

Key Features

  • Layer Stack: Drag and drop up to 5 audio samples as colored visual layers with 3D perspective view
  • Output Curve: Draw a golden target amplitude envelope — the combined output matches this shape exactly
  • Priority Mix: Highest priority layer plays naturally, lower layers complement to fill the curve
  • Frequency Zones: Each layer owns a frequency band (bass, lows, mids, highs, air) for spectral clarity
  • Layer Editor: Ableton-style clip editor with waveform, warp markers, pitch, loop points, and per-layer EQ
  • Per-Layer EQ: Parametric EQ with spectrogram visualization and draggable nodes
  • Match Gauge: Real-time percentage showing how closely output follows the target curve
  • Contribution Meters: Per-layer horizontal bars showing each layer's contribution to the mix

System Architecture

Audio Samples (drag & drop) ↓ Layer Stack (up to 5 layers, colored, reorderable) ↓ Priority Engine ├── Layer Priority (volume/headroom order) └── Frequency Priority (spectral ownership zones) ↓ Output Curve Matching (target envelope → auto-mix) ↓ Per-Layer Processing ├── Waveform Editor (warp, pitch, loop) └── Parametric EQ (per-layer sculpting) ↓ Master Output (match gauge, contribution meters)

Why I Built This

As a producer, layering samples is fundamental but the mixing process is tedious — constantly adjusting volumes, carving EQ, and hoping the layers complement each other. LAYERS flips the workflow: instead of tweaking individual layers to hope for a good output, you define the output you want and let the engine figure out how each layer contributes. The priority system ensures your most important layer (e.g., a kick's low-end) never gets compromised.
