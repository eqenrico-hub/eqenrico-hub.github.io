What It Does

A comprehensive platform for analyzing Freedom of Information (FOI) request data across multiple jurisdictions. Aggregates, analyzes, and visualizes FOI responsiveness patterns.

Problem It Solves

FOI data is scattered across different platforms with no unified analysis. This consolidates multiple data sources to provide actionable intelligence on public sector transparency.

Key Features

  • Multi-source FOI data aggregation
  • Responsiveness scoring algorithms
  • Trend analysis and forecasting
  • Cross-jurisdictional comparison
  • Research-ready data exports

Tech Stack

Python PostgreSQL Streamlit Pandas

Why I Built This

Central to my PhD research on using FOI responsiveness as a proxy for public sector accountability. This platform enables rigorous academic analysis of transparency patterns.
