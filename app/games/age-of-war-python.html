What It Does

A faithful recreation of the beloved Flash game "Age of War" built in Python. Command your base through different ages of history, spawning units and upgrading your civilization to defeat the enemy.

Problem It Solves

With Flash being discontinued, many classic browser games became unplayable. This project preserves the Age of War experience as a standalone Python application.

Key Features

  • Multiple historical ages with unique units
  • Base defense and upgrade system
  • Special abilities and turrets
  • Progressive difficulty
  • Original game assets

Tech Stack

Python Pygame

Why I Built This

Age of War was one of my favorite Flash games growing up. When Flash died, I wanted to preserve the experience and learn game development in Python at the same time.
