What It Does

A React-based chess training app focused on learning and memorizing chess openings. Uses spaced repetition to help you retain opening lines long-term.

Problem It Solves

Chess openings are notoriously hard to memorize - there are thousands of variations. Most players forget lines they've studied within days. This app uses proven memory techniques to make opening knowledge stick.

Key Features

  • Interactive Board - Practice moves on a real chessboard
  • Spaced Repetition - Smart scheduling of review sessions
  • Opening Database - Common openings with variations
  • Progress Tracking - See which lines you know well
  • PDF Export - Generate study materials

Tech Stack

React TypeScript chess.js Vite Firebase

Why I Built This

As a chess player, I was frustrated that I kept forgetting openings I'd studied. Existing tools were either too complex or didn't use effective learning techniques. I wanted something simple that actually works.
