What It Does

A browser-based logic puzzle game where players solve tautological puzzles across 5 unique worlds. Features a story mode with cinematics, progressive difficulty, and boss battles.

Problem It Solves

Most logic games are either too dry (pure puzzles) or too casual (no real challenge). Truth Forge combines rigorous logical thinking with engaging storytelling and game mechanics.

Key Features

  • 5 Unique Worlds: Logic Academy, Mystery Mansion, Space Station, Enchanted Forest, Time Paradox Tower
  • 25 Levels with progressive difficulty
  • Story cinematics with dialog system
  • Boss battles at world ends
  • Progressive unlocking system

Tech Stack

JavaScript Phaser.js Python (backend)

Why I Built This

I wanted to make formal logic accessible and fun. The game teaches logical reasoning through gameplay rather than dry exercises.
