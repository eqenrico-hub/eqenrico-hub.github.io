What It Does

A dedicated score tracking app for Skull King, the pirate-themed trick-taking card game. Handles all the complex scoring rules automatically.

Problem It Solves

Skull King has complicated scoring with bonuses for exact predictions, capturing pirates, etc. Manual scorekeeping is error-prone and slows down the game.

Key Features

  • Automatic score calculation
  • Bonus tracking (pirates, mermaids, etc.)
  • Round-by-round history
  • Multiple player support
  • Game statistics

Tech Stack

React Tailwind CSS Vite

Why I Built This

We kept making scoring mistakes during game nights. This app eliminates arguments about who captured what and keeps the game moving.
