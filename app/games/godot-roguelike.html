What It Does

A roguelike dungeon crawler featuring procedurally generated levels, permadeath, and a variety of enemies and items. Built with the Godot game engine.

Key Features

  • Procedurally generated dungeons
  • Permadeath with persistent unlocks
  • Multiple character classes
  • Item and ability system
  • AI-generated scenario assets

Tech Stack

Godot Engine GDScript

Why I Built This

Roguelikes are my favorite genre, and I wanted to learn Godot by building something I'm passionate about. The procedural generation keeps every run fresh.
