What It Does

A web-based version of Age of War that runs directly in the browser. Features original art assets and sounds extracted from the classic Flash game.

Problem It Solves

Makes the classic Age of War experience accessible on any device with a web browser, no installation required.

Key Features

  • Runs in any modern browser
  • Original art and sound assets
  • Mobile-responsive controls
  • No installation needed
  • Music: "Glorious Morning" by Waterflame

Tech Stack

HTML5 JavaScript Canvas API

Why I Built This

After building the Python version, I wanted a version that anyone could play instantly in their browser without downloading anything.
