Agents

Autonomous tools that orchestrate multi-step workflows — calling APIs, analyzing results, retrying on failure, and delivering finished outputs.

🤖 MapBuilder

Autonomous fantasy world generator. Takes a single text prompt ("dark forest", "cyberpunk alley") and orchestrates the full pipeline: generates a Wang tileset via PixelLab API, auto-detects tile corner patterns, validates completeness (retries up to 2x if tiles are missing), fills gaps by flipping existing tiles, optionally generates 10 matching buildings, then opens a map editor pre-loaded with the new world.
