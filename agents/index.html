Agents

Autonomous tools that orchestrate multi-step workflows — calling APIs, analyzing results, retrying on failure, and delivering finished outputs.

🎨 Blender Redesign Agent

Bridges ChatGPT (or any vision-LLM) to Blender for rapid GUI iteration. Renders a tight top-down view of one GUI section with numbered yellow badges, provides a prompt template that forces a parseable mm-coordinate table, and rebuilds the geometry in Blender from ChatGPT's table in one click. Built for VST plugin GUI work — the coord table survives even when the LLM can't generate images.

🖼️ GUI Frame Tool

Browser-based 2D editor for translating Blender GUI layouts to draggable frames. Load a layout JSON (exported from Blender), drag/resize any frame with 8 handles (corners + edges), Shift-snap to align with other frames, add new frames, delete frames, attach comments that Claude reads when applying changes back to Blender. Auto-fits to any window size. The round-trip tool for tuning VST GUI proportions before committing to 3D.

🤖 MapBuilder

Autonomous fantasy world generator. Takes a single text prompt ("dark forest", "cyberpunk alley") and orchestrates the full pipeline: generates a Wang tileset via PixelLab API, auto-detects tile corner patterns, validates completeness (retries up to 2x if tiles are missing), fills gaps by flipping existing tiles, optionally generates 10 matching buildings, then opens a map editor pre-loaded with the new world.
