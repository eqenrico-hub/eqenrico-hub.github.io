Active clone layer (auto-detected):
Any region pressed → paint goes to the Pressed layer.
All released → paint goes to the Unpressed layer.
Pixels OUTSIDE the size X / size Y boundary where stamp alpha falls off to 0. Inside = full opacity.
Alt+click to set source · drag to paint over with cloned pixels
Linked clone (visible only when one specific button is pressed):
Strokes go to that region's private clone layer. Render path: drawn ONLY when the linked region is pressed.
Fade brush — softens the active clone-stamp layer wherever you drag.
Drag to gradually erase whatever you see in preview (decals, clone stamps, etc) with a soft falloff. Down to 0.3 px for tight edge feathering.
Paint brush — draws on top of everything in the preview (decals, clones, etc).
Blur brush — softens whatever you see in preview by sampling the rendered pixels and blending them.
Strength = how much the blurred sample blends in over each stamp. Multiple drags accumulate.

4. Decal overlay

Upload a transparent PNG (e.g. a metal ring). It becomes a draggable/resizable region that renders the image at its bbox.

4b. Text label

Drops a text region at canvas centre. In the region inspector you can edit content, font, size, colour, link to a parent button, and offset its position when the parent is pressed.

5. Import / Export

Each region → 1 PNG cropped from B at native resolution. JSON manifest maps each region to {name, bbox, crop_file}.
