🎨 Blender Redesign Agent

ChatGPT → Blender recreation pipeline for GUI sections

What it does

Given any Blender GUI section (e.g. a VST plugin panel), the agent:

  1. Renders the section top-down with numbered yellow badges on every interactive element.
  2. Emits a prompt to feed ChatGPT (or any vision-LLM) that explicitly requests a coordinate table in mm.
  3. Parses ChatGPT's returned table and rebuilds the geometry in Blender at the new coordinates.

One-command workflow

python3 run.py scalechord     # or: eq, header, carrier_mix, main_knobs, right_knobs, band_display
# Outputs:
#   /Users/ricosan/Downloads/scalechord_render_in_blender.py   (run in Blender)
#   /Users/ricosan/Downloads/scalechord_prompt.txt             (send to ChatGPT)

Files

  • run.py — unified runner. Takes a section key, emits Blender render script + pre-filled prompt.
  • sections.json — registry of all Band Ritual sections with their Blender object names and labels. Add new sections here.
  • prompt_template.txt — Prompt blueprint with placeholders. Designed to force a parseable coord table even when image gen fails.
  • overlay_numbers.py — Burns numbered badges onto the rendered PNG.
  • validate_table.py — Validates ChatGPT's table response against the section spec before Blender apply. Catches duplicates, hallucinated elements, out-of-bounds coords.
  • apply_design.py — Paste ChatGPT's table into the TABLE variable, run in Blender, geometry rebuilds.

Workflow

1. Blender:  run render_section.py    → section_render.png + /tmp/section_overlay.json
2. Host:     python3 overlay_numbers.py → numbered badges burned in
3. ChatGPT:  upload PNG + paste prompt_template.txt (filled in)
4. Copy table from response → paste into apply_design.py TABLE variable
5. Blender:  run apply_design.py      → new geometry at ChatGPT's coords

Why a TABLE not just an image

Text-only ChatGPT models can't generate PNGs. The agent is built so the coordinate table is the primary output — it's all that's needed to rebuild in Blender. The image is optional.
