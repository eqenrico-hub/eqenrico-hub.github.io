🎨 Moodboard Generator

Extract style DNA from reference VSTs into a structured moodboard

What it does

Given a current plugin GUI (any DAW screenshot) + one or more style-reference VSTs, this prompt asks a vision-LLM (ChatGPT / Gemini / Claude) to produce a single moodboard image containing palette swatches with hex codes, material samples, typography specimens, decorative motifs, and lighting vignettes — all distilled from the style references.

The moodboard is not a UI mockup — it's an abstract visual style guide you can reference when rebuilding the GUI in Blender or HTML.

How to use

  1. Open prompt.txt in a text editor, copy all content.
  2. In ChatGPT (or similar), start a new conversation.
  3. Upload IMAGE 1: your plugin's current GUI screenshot.
  4. Upload IMAGE 2+: one or more style reference VST screenshots.
  5. Paste the prompt as your message.
  6. The model returns a moodboard PNG.

When this is useful

  • Before committing to a full redesign — see what the "style DNA" of the references actually is.
  • As a reference sheet to share with teammates or iterate separately.
  • As input to a subsequent GUI-redesign prompt (though typically you can skip straight to redesign — the moodboard step is optional).
